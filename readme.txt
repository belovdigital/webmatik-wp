Run AI-powered website audits directly from your WordPress dashboard. Get a Growth Score and prioritized action plan.

== Description ==

Webmatik analyzes your website across 50+ growth signals in 7 categories:

* **Performance** — Core Web Vitals, load speed, mobile experience
* **SEO** — Technical SEO, keywords, structured data, meta tags
* **AI Search (GEO)** — Visibility in ChatGPT, Gemini, Perplexity
* **UI/UX** — Design, layout, navigation, mobile responsiveness
* **Conversion** — CTAs, trust signals, forms, value proposition
* **Retention** — Engagement patterns, return visit signals
* **Accessibility** — WCAG checks, keyboard navigation, screen readers

Each audit produces a Growth Score (0–100) and a prioritized action plan with step-by-step fix instructions grouped by effort level.

**How it works:**

1. Install this plugin
2. Click "Connect to Webmatik" — sign in or create a free account
3. Authorize the connection (one click)
4. Click "Run Audit" — results appear in 2–3 minutes

**Features:**

* One-click audit from WordPress dashboard
* Growth Score widget on WP Dashboard
* Links to full interactive report on webmatik.ai
* Uses your existing Webmatik plan and audit quota

== Installation ==

1. Upload the `webmatik` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Webmatik in the admin menu
4. Click "Connect to Webmatik" to authorize
5. Click "Run Audit"

== Frequently Asked Questions ==

= Do I need a Webmatik account? =

Yes. You can create a free account directly from the plugin via the "Connect to Webmatik" button. The free plan includes 3 audits per month.

= What data does the plugin send? =

Only your site URL is sent to the Webmatik API. The plugin does not collect or send any other data from your WordPress installation.

= How long does an audit take? =

Typically 2–3 minutes. The plugin polls for results automatically.

= Does this affect my site's performance? =

No. The audit runs on Webmatik's servers, not on your WordPress installation.
