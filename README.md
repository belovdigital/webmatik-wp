## ✨ Features

- **🚀 One-Click Audits** — Run a full AI-powered Growth Audit without leaving WP admin
- **📊 Growth Score Widget** — See your site's score and grade right on the WordPress Dashboard
- **🔗 Full Report Link** — Jump to the interactive report on [webmatik.ai](https://webmatik.ai) for deep analysis
- **🔐 Secure OAuth-like Connection** — Connect your Webmatik account via a popup flow — no manual API keys
- **📱 7 Audit Categories** — Performance, SEO, AI Search (GEO), UI/UX, Conversion, Retention, Accessibility
- **🎯 50+ Growth Signals** — Every audit evaluates over 50 signals and produces a prioritized action plan
- **💡 Actionable Recommendations** — Step-by-step fixes grouped by effort level (quick wins → deep dives)
- **🆓 Free Plan Available** — Get started with 3 free audits per month

## 📦 Installation

### From WordPress Admin (recommended)

1. Go to **Plugins → Add New** in your WordPress admin
2. Search for **"Webmatik"**
3. Click **Install Now**, then **Activate**
4. Navigate to **Webmatik** in the admin sidebar
5. Click **Connect to Webmatik** to link your account

### Manual Installation

1. Download the plugin ZIP from [WordPress.org](https://wordpress.org/plugins/webmatik-ai-website-audit/) or this repository
2. Upload the `webmatik` folder to `/wp-content/plugins/`
3. Activate the plugin via **Plugins → Installed Plugins**
4. Go to **Webmatik** in the admin sidebar and connect your account

## 🔧 How It Works

```
┌─────────────────┐       ┌───────────────┐       ┌─────────────────┐
│  1. Connect      │──────▶│  2. Run Audit  │──────▶│  3. Get Results  │
│                  │       │                │       │                  │
│  Click "Connect" │       │  Click "Run    │       │  Growth Score,   │
│  to authorize    │       │  Audit" — your │       │  grade, and link │
│  your Webmatik   │       │  site URL is   │       │  to full report  │
│  account via     │       │  sent to the   │       │  on webmatik.ai  │
│  secure popup    │       │  Webmatik API  │       │                  │
└─────────────────┘       └───────────────┘       └─────────────────┘
```

1. **Connect** — Click "Connect to Webmatik" to open a secure popup. Sign in or create a free account, then authorize the connection with one click.
2. **Run Audit** — Click "Run Audit". Your site URL is sent to the Webmatik API, which analyzes it across 7 categories (typically 2–3 minutes).
3. **View Results** — Your Growth Score and grade appear in the plugin page and on the WP Dashboard widget. Click through to the full interactive report on [webmatik.ai](https://webmatik.ai).

## ❓ FAQ

<details>
<summary><strong>Do I need a Webmatik account?</strong></summary>

Yes — you can create one for free directly from the plugin by clicking "Connect to Webmatik". The free plan includes **3 audits per month**.
</details>

<details>
<summary><strong>What data does the plugin send?</strong></summary>

Only your **site URL** is sent to the Webmatik API. The plugin does not collect or transmit any other data from your WordPress installation — no posts, users, or database content.
</details>

<details>
<summary><strong>How long does an audit take?</strong></summary>

Typically **2–3 minutes**. The plugin automatically polls for results and updates the UI when the audit is complete.
</details>

<details>
<summary><strong>Does this affect my site's performance?</strong></summary>

No. The audit runs entirely on **Webmatik's servers**, not on your WordPress hosting. Your site visitors are not affected.
</details>

<details>
<summary><strong>What is a Growth Score?</strong></summary>

A score from **0 to 100** that reflects how well your website is optimized across all 7 categories. Grades range from Poor to Excellent.
</details>

<details>
<summary><strong>What is GEO / AI Search?</strong></summary>

**GEO (Generative Engine Optimization)** measures how visible your website is to AI-powered search engines like ChatGPT, Gemini, and Perplexity. As more users search via AI, GEO is becoming a critical growth channel.
</details>

<details>
<summary><strong>What WordPress versions are supported?</strong></summary>

WordPress **5.8+** with PHP **7.4+**. Tested up to WordPress 6.9.
</details>

<details>
<summary><strong>Can I run audits on a staging or local site?</strong></summary>

The audited URL must be **publicly accessible** — Webmatik's servers need to reach it to perform the analysis.
</details>
